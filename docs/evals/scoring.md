# Evaluation Scoring

Deck scores are diagnostic, not proof that a deck is good.

Current dimensions:

- Curve
- Mana
- Interaction
- Synergy
- Resilience
- Bracket fit

Game evaluations should log:

- opening hand keep/mulligan
- missed land drops
- unused mana by turn
- removal timing
- threat assessment accuracy
- game-ending lines seen or missed
- final placement

Use post-game critique to update Markdown knowledge and deck recommendations.
